# TrackFit Mini

**Release:** 1.0.0

## Descrição
Projeto React + TypeScript simulando um diário de treino com três componentes:
- Welcome – Mensagem de boas-vindas
- ExerciseList – Lista de exercícios
- MotivationalMessage – Frase motivacional com props tipadas
